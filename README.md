Конвертация валют

Проект на React c TypeScript

- Получение списка курса валют из отдельного API
- Конвертация одной валюты в другую и наоборот (без бэкенда)

Использовался: 
- Хук useState 
- Хук useEffect
- Хук useRef
- Условный рендер
- Работа с Api
